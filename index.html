Robots Blinking Eyes using GC9A01 Display

This project implements realistic blinking eyes for a robot using two GC9A01 1.28-Inch Round LCD TFT Displays. While originally inspired by "Uncanny Eyes" for ESP32, this version is optimized for the Raspberry Pi Pico (RP2040).

The Optimization: Because the RP2040 lacks the memory to render full-frame eye animations in real-time at high speeds, this solution writes a BMP image to the display buffer at boot. It then dynamically updates only specific parts of the display to move the pupils and trigger blinks, resulting in a smooth, high-speed effect.

Interactive Capabilities

The eyes are designed to be part of a larger robotic ecosystem. By integrating an ESP32-CAM, the system can receive X-Y coordinates of a person's face, allowing the eyes to track and follow individuals across a room.

In the current repository code, the eyes move to random positions—perfect for a robot head that is already physically panning to follow a subject.
